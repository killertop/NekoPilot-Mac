# Changelog

本文件记录 NekoPilot for Mac 的用户可见变化。开发中的变更先记录在当前版本的 Unreleased 区域，发布时再整理为正式版本。

## Unreleased

### 产品基线

- 建立 NekoPilot for Mac 的 macOS-first Tauri + React + Rust 应用基线。
- 统一产品名称、窗口标题、托盘提示、深链 Scheme 和 Bundle ID。
- 支持订阅配置与独立节点链接的导入和管理。
- 支持节点选择、Rule / Global 路由模式和连接执行状态反馈。
- 增加 Rust、前端和配置相关的单元测试覆盖。

### 发布与工程

- 增加本地 Release 构建和 macOS bundle 说明。
- 增加正式签名、公证和 GitHub Actions 发布说明。
- 整理中英文项目文档、开发指南、安全说明和贡献指南。
