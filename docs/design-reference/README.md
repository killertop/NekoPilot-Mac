# Compact UI visual reference

These screenshots preserve the compact 371 × 600 NekoPilot layout used as a visual regression reference during the native SwiftUI migration.

They define product proportions such as card density, page spacing, bottom navigation, modal width, typography hierarchy, and the single-column macOS panel shape. They do not define an implementation technology or current data model.

When changing SwiftUI views, compare a current screenshot against the matching reference at the same viewport and state. The live SwiftUI source and current product requirements remain authoritative when content or behavior has changed.
